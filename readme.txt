Enables you to display how many times a post/page had been viewed by user or bot.

== Description ==
It can set that if count the registered member views.
To differentiate between USER and BOT is by HTTP_agent, and it can set at admin

== Installation ==
1. Go to `WP-Admin -> Plugins -> Add New` to search WP-PostViews Plus.
2. Pass `Install Now` to install the plugin
3. Activate `WP-PostViews Plus` Plugin
4. Go to `WP-Admin -> Settings -> PostViews+` to configure the plugin.
